# Changelog

All notable changes to this project will be documented in this file.

## [0.0.2] - 2026-05-08

- Added a `.gitignore` file to exclude development artifacts such as `node_modules`, build output, generated `.vsix` packages, local VS Code settings, and log files.
- Added a default **Run Extension** launch configuration under `.vscode/launch.json` to make debugging the extension easier.
- Updated `package.json` to include repository metadata, an explicit `MIT` license, and a `categories` entry.
- Added this `CHANGELOG.md` to document project history.

## [0.0.1] - 2026-05-08

- Initial release with basic extension scaffold registering the `Agent Lens: Run Preview` command.
